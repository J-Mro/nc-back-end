# NC News Seeding

# Project Dependencies

Run npm install to install the necessary dependencies

# Database set-up

Run `npm run setup-dbs` to create your database<br/>

# Environment Variables

Create .env files for your development and test databases

- These files will be of the form:
- `PGDATABASE = <your-database-name>`

# Seeding your database

Run `npm run seed-dev` to seed your database
